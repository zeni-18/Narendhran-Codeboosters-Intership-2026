# Day 01: Introduction to Data Engineering - Homework & Classwork

Welcome to the repository for Day 01. This directory contains Python scripts and datasets analyzing student performance and housing data using the Pandas library.

---



## 🚀 How to Run the Files

Make sure you have `pandas` installed:
```bash
pip install pandas
```

You can run any of the python scripts from this directory:
```bash
# Run the Mini Project Report
python Mini_projects.py

# Run the Practice Questions Solutions
python practice_questions.py

# Run Classwork Code
python classwork.py
```
